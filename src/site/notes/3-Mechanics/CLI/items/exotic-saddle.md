# Exotic Saddle
*Tack and harness*  


- **Cost**: 60 gp
- **Weight**: 40.0 lbs.

An exotic saddle is required for riding any aquatic or flying mount.

*Source: Player's Handbook p. 157. Available in the <span title='Systems Reference Document (5.1)'>SRD</span> and the Basic Rules (2014)*
